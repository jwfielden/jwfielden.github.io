Project Idea

Project Overview

To create a smartphone App that helps with your finances, calendar, to-do-list and time planner. An all in one App that can be shared with a friend or family member, can see where you are paying off bills (with dates), link bank accounts to the app and easy creation of tasks so you can help balance out priorities from important to least important.

Motivation for Project

Apps that manage your finance currently such as Mint and Wally, focus on just the budget management part, checking your bills and and budget checker, that inspects the budget you have input and money being spent and you purchase goods or services the money can be input however it may need to be done manually and may need the use of multiple third party budget apps or calendars to complete the cycle of budget and finance control.

Description of Project

The user can start the process by downloading the app and once downloaded can start by entering all personal details, number of bank accounts, bills either weekly or monthly, the app will give the option if it's either a once off or continuous payment. A built in calendar (monthly inspection) that gives the user the ease of use option to check the current day and when the next payments will come out. The app will need to inspect the banking apps used and acquire data from how much money is currently in the bank account so it can record and recognize spending habits and produce a graph of monthly and yearly spendings.

This service will feature a Simple user interface with hamburger button in the corner to start adding and compiling information that will help the user start the process with adding in bank account details, listing all expenditures. Once the information has been added the user will be greeted for more drop down menu options in the Hamburger button menu. Graphs that indicate how someone is spending or even showing savings. A list of every single expenditure that the user has added and the calendar option the user can inspect on a weekly/monthly to see a highlight of future or past bills that have been or need to be paid. Over time the user will be able to find relevant patterns that they can adjust over time and continuously make modifications depending on their buying and saving style.

Tools, Tech and Skills Required

The program will be developed with Visual Studio Code may be the basis used to develop this application. As this app will be on a phone and connected to the internet, the program will need to be made so it can be inspected on the internet using a computer. HTML will be the foundation for creating websites so that the code made for the app version can be converted for the website version.

The coder will need to have an understanding of the basics of front-end and back-end development, potentially using programming languages such as Java, Swift, Xamarin or C# and using Visual Studio Code to make the app version. Calculators will be made specific to this app where you can put in data and calculate certain times or dates to pay off or save up for a bill.

Outcome

If this project does become successful, it will be able to support many people who are wanting alternative options trying to save and budget properly in an all in one ecosystem where they don't need to rely on multiple apps to record this data. The user of this app will have all their personal money data recorded and will be able to organise all their expenses and also see how much they are saving so it makes ease of use for budgeting a more user friendly option instead of having multiple apps and external calendars to rely on and is conveniently is stored in one location.
